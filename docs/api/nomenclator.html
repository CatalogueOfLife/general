NAV

    Unified Nomenclator API (vv1)

    https://api.col.plus

    Introduction

    The unified nomenclator developed by Catalogue of Life plus tracks scientific names across all life and ties them to literature and types. It tracks recombinations and provides a place for collaborative editorial efforts and review.

    Authentication

    Basic Authentication

    Supported for authenticating all API requests.

    OAuth 2.0

    Supported for authenticating all API requests.

    Header

    Name Type Required Description
    Authorization string yes

    Used to send a valid OAuth 2 access token. Do not use with the “access_token” query string parameter.

    Parameters

    Name Type Pattern Required Default Description
    access_token string yes

    Used to send a valid OAuth 2 access token. Do not use with the “Authorization” header.

    Response 401

    Bad or expired token. This can happen if the user or system revoked or expired an access token. To fix, re-authenticate the user.

    Response 403

    Bad OAuth request (wrong consumer key, bad nonce, expired timestamp…). Unfortunately, re-authenticating the user won’t help here.
